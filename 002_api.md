## Endpoints

### Videos

#### `GET /videos/top-buzz`
Returns the top-ranked videos by Buzz Score.

**Query params:**
| Param | Type | Default | Description |
|---|---|---|---|
| `limit` | int | 20 | Number of results |
| `offset` | int | 0 | Pagination offset |
| `min_score` | float | 0 | Minimum Buzz Score |

**Response:**
```json
{
  "updated_at": "2024-03-17T09:00:00Z",
  "total": 100,
  "results": [
    {
      "rank": 1,
      "video_id": "abc123",
      "title": "Why Gen Z hate working overtime",
      "channel_name": "WorkLife TV",
      "channel_id": "UC...",
      "thumbnail_url": "https://i.ytimg.com/...",
      "view_count": 320000,
      "like_count": 14000,
      "comment_count": 8200,
      "publish_time": "2024-03-10T12:00:00Z",
      "buzz_score": 82.4,
      "fresh_score": 0.13,
      "view_velocity": 45714,
      "viral_spike": false
    }
  ]
}
```

---

#### `GET /videos/top-discussion`
Returns the top-ranked videos by Discussion Score.

**Query params:** same as `/top-buzz` plus `min_debate_ratio` (float, default 0)

**Response:** same shape, replaces `buzz_score` with:
```json
{
  "discussion_score": 91.2,
  "debate_ratio": 0.38,
  "comment_density": 0.026,
  "comment_velocity": 1171.4
}
```

---

#### `GET /videos/viral`
Returns videos flagged as viral spikes.

**Query params:** `limit`, `offset`

**Response:** same shape, includes:
```json
{
  "view_velocity": 45714,
  "channel_avg_velocity": 12000,
  "spike_ratio": 3.8,
  "viral_spike": true
}
```

---

### Trends

#### `GET /trends/topics`
Returns detected topic clusters with trend velocity.

**Query params:**
| Param | Type | Default | Description |
|---|---|---|---|
| `status` | string | `all` | Filter: `emerging` or `all` |
| `limit` | int | 20 | Number of topics |

**Response:**
```json
{
  "updated_at": "2024-03-17T09:00:00Z",
  "topics": [
    {
      "id": 1,
      "topic_name": "Gen Z work culture",
      "keywords": ["Gen Z", "overtime", "work-life balance"],
      "trend_score": 78.5,
      "trend_velocity": 4.2,
      "status": "emerging",
      "video_count": 14
    }
  ]
}
```

---

### Channels

#### `GET /channels/discovered`
Returns channels discovered via related-video expansion.

**Query params:** `limit`, `offset`, `source` (`curated` | `discovered` | `all`)

**Response:**
```json
{
  "channels": [
    {
      "id": 1,
      "channel_id": "UCxxx",
      "name": "TechTalks",
      "subscriber_count": 250000,
      "is_curated": false,
      "avg_buzz_score": 67.3,
      "videos_crawled": 12,
      "discovered_at": "2024-03-15T08:00:00Z"
    }
  ]
}
```

---

### Internal / Admin (not public-facing)

#### `POST /tasks/trigger-crawl`
Manually trigger the daily crawl (admin use).

#### `GET /health`
Returns service health status and last task completion times.

```json
{
  "status": "ok",
  "last_crawl": "2024-03-17T04:00:00Z",
  "last_ranking": "2024-03-17T05:00:00Z",
  "videos_in_db": 4821,
  "channels_in_db": 87
}
```

## Error Format
```json
{
  "error": "NOT_FOUND",
  "message": "No rankings found for today. Crawl may not have run yet.",
  "timestamp": "2024-03-17T10:00:00Z"
}
```

## HTTP Status Codes
| Code | Meaning |
|---|---|
| 200 | Success |
| 400 | Invalid query params |
| 404 | Resource not found |
| 429 | Rate limit exceeded |
| 500 | Internal server error |
